========================================================================
    静态库 : PokemonRom 项目概况
========================================================================

应用程序向导已为您创建了这个 PokemonRom 库项目。
此文件包含组成 PokemonRom 应用程序
的每个文件的内容摘要。


PokemonRom.vcproj
    这是用应用程序向导生成的 VC++ 项目的主项目文件。
    它包含有关生成此文件的 Visual C++ 版本的信息，以及
    有关用应用程序向导选择的
    平台、配置和项目功能的信息。


/////////////////////////////////////////////////////////////////////////////

StdAfx.h、StdAfx.cpp
    这些文件用于生成名为 PokemonRom.pch
    的预编译头(PCH)文件以及名为 StdAfx.obj 的预编译类型文件。

/////////////////////////////////////////////////////////////////////////////
编译器和链接器开关经过了修改，可以支持 MFC。对此项目使用
MFC ClassWizard 要求向项目添加若干
文件，包括 "resource.h"、"PokemonRom.rc" 和 "PokemonRom.h"(其中
包含 resource.h)。如果向静态库添加 rc 文件，则可能
遇到麻烦，因为存在这样的限制，即只有一个 rc 文件可以
出现在 Dll 或 Exe 中。通过将
库的 .rc 文件添加到父项目的 .rc 文件中，可以解决此问题。
